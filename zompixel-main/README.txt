STATIC APK WEBSITE (NO NPM)

1) Put your APK in the apk folder.
   Default filename used by the page:
   apk/app-release.apk

2) Open index.html to view the project page.

3) Share your hosted URL for normal download button usage.

4) Optional: use custom APK filename from URL query.
   Example:
   ?file=myapp-v2.apk
   Then the download button points to:
   apk/myapp-v2.apk

5) Optional screenshot:
   Add preview.png in root folder to show screenshot card image.

IMPORTANT:
- This is static only. No backend upload API.
- For other people to download, host this full folder on static hosting.
